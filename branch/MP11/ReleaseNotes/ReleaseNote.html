TVSeries v2.6 Release Highlights
Import Process

Improved Import Process in configuration

Importing in configuration has had a major overhaul, its now a wizard based process.

Users can change the local file information before import begins in the wizard e.g. edit the episode number or season number if tvseries does not recognise it.

No more annoying popup dialogs awaiting series approval or episode order approval. This is now done as part of the wizard.

Wizard also allows Local File to Online Episode Matching, e.g. Aired, DVD Order, Title matching or users can manually match an episode from the list.

Improved feedback of Import Process, users can see a breakdown of the entire import procedure instead of a single progress bar.

Improved Default Regular Expression, so now even more files are automatically recognized. Click Reset in 'Regular Expression' page in configuration if upgrading.

Improved DVD Ordering such that it uses the Aired Order if DVD values are not set online.

Forcing a new online match for a series online will now always prompt to choose series.

Users can now change the 'Local File to Online Matching Method' for a series from the GUI in the Action Menu. This allows your files to be rematched against different ordering such as DVD Order.

Users can now sort the episode view by DVD Ordering or Aired Ordering.

Note: Due to changes above, users whom already have episodes imported by DVD order need to delete these series from database and re-import.

User Edits

User modifications on database now persist

You can now edit a field in the database via the Details screen in configuration and have its data persist. i.e. it will not be overwritten when an online update is run. For instance you can modify the Genres or a Series Title to your preference.

Changes done to fields in database are highlighted with a different colour in configuration.

To restore the default data for a field, simply delete the contents and it will automatically restore.

Play Behaviour

Pressing 'Play' button on a series now has a whole range of play options

Configuration 'General' tab, allows user to modify the behaviour of the 'Play' button when a series is selected. User can set to play the first unwatched episode, random episode, latest episode etc.

User can also configure to display a menu of choices when 'Play' is pressed.

New Episodes Indicator

New Episodes for a series can easily be identified by user in banner views

Configuration allows user to display a 'New' image icon on a banner/poster in 'WideBanners' and 'Filmstrip' layouts.

Behaviour of image can either be changed from 'Recently Added Episodes' or 'Unwatched Episodes'. This can be changed in the 'General' tab in configuration. Option can also be disabled

New Skin Settings for skinner to control the position of the image on a thumbnail.

If setting is enabled, but skin does not support it. A text style image is overlayed on banner/poster.

Note: see Blue3Wide for reference settings and image

Subtitle Central Support

Support for Subcentral has been added to TVSeries

Sub-Central for TVSeries can be enabled in the 'Extras' tab in configuration.

Note: the 'Sub-Central' plugin must be installed, see 'Sub-Central' plugin for MediaPortal for more information.

Video Play and OSD Images

New Image support for OSD and custom Skin Properties on Play

Skinner can now change the image pushed to the skin in OSD. This can be configured in TVSeries Skin Settings. Series Poster, Season Poster, Episode Image or even a custom image can be displayed.

New 'Play' properties can be created in Skin Settings e.g. skinner can create a property to display 'ClearArt' as seen in some XBMC skins on the OSD.

Note: see Blue3Wide for reference settings

General / Misc

New option to display full filename in dialog when choosing subtitle to download.

Updated MediaInfo support to include new fields, better support/detection for HD codecs. It's recommended users Update MediaInfo using the link in the 'Details' tab in configuration. Skinners may need to be aware of some possible changes to some fields.

Fixed issue with importer failing to detect move/copy of folders to local import paths.

Full details of episode played in playlist view will now be shown in OSD.

Added Watched/Unwatched icons to playlist view.

New setting in configuration to add only 'Unwatched' episodes to a playlist.

When removing a custom view, all series associated with that view are now removed also.

Renamed 'Force Online Match' has been renamed to better describe its function.

Improved support for user defined scripts before/after playback.

Added Regular Expression support in 'String Replacements' section.

TVSeries v2.3 Release Highlights
Views System

The views system has been upgraded to support several new features:

The user can now create tagged views, this allows you to add any series to a defined view e.g. Documentaries, Kids Shows, Adults Shows etc...

Tagged Views can be created when adding a series to a new view from the plugins context menu with-in MediaPortal or from the Views tab in Configuration. The Configuration program allows you to quickly multi-select a range of Series to add to a particular view using the Add/Edit view dialog.

Several new Built-in views have been added, some of which include: Actors, Adult Series, Kids Series, Continuing Series, Ended Series, Top 10 etc. These views are available for selection in the Views Configuration using the Templates dialog.

Parental Control support, you can now protect views from the configuration views tab. After you have locked down your views, enter a 4-Digit Pin in the Pin Code dialog. When a protected view is selected from the plugin in MediaPortal you will be prompted to insert your Pin Code, if the pin entered is correct you will see the contents of the protected view. If you have correctly entered a pin code and then re-enter the view you will not be prompted unless you restart/resume or manually lock views from the context menu.

Rate Series/Episodes

A few new features have been added to enhance the users experience when rating episodes or series:

Supported skins will now show a Graphical Rating dialog, this will also help you rate an episode or series by giving you a meaningful description for the score e.g. Mediocre, Good, Great. Thanks goes to the MovingPictures team for sharing the dialog.

The User can choose between a 5 Star or a 10 Star rate dialog from Configuration under General Settings.

Fixed an issue where on occasion the episode list will disappear after rating an episode.

Import Settings

The Online Data settings in configuration have been updated to provide several new features:

You can now turn off Import when starting/resuming MediaPortal, this is of particular interest to people experiencing performance issues or have slow/non-existant internet connections. To completely turn off automatic imports you also need to disable Updates.

Downloading of user ratings can be disabled, when disabled the import will be faster but your episode ratings will not be synced with the online database.

Downloading of missing artwork can be disabled, during import the plugin will check if any series or season has no artwork available. If no artwork is found then an attempt will be done to check if any is available online and download them. Import will be faster when disabling, time difference is neglible on smaller databases.

During import the plugin checks all series fanart and updates the local database with any missing or new fanarts. You can now disable this option for a faster import if you only want check for updates.

You can now check for updates for episode(s) and series from Details tab in configuration. This will only do the updates on the selected item, it will not do a full import.

Automatic download of Fanart that contain Series Names/Logos is now optional, this is turned off by default.

Installer Package

An Installer has been created to help with deployment of the plugin, it has the following features:

Support for all platforms supported by MediaPortal and is also compatible with the latest MediaPortal 1.1.0 release.

Custom directories defined in MediaPortalDirs.xml will be recognised and used for all components e.g. plugin, languages, skins etc

The installer supports upgrades for future releases.

The plugin and all of its components can be uninstalled from the Add\Remove programs in the Windows Control Panel.

During Installation the database is backed up the users documents directory: %USERPROFILE%\Documents\MP-TVSeries.

Options to remove artwork and database after an Uninstall to help keep your system clean.

Checks for multiple instances of the plugin after installation, this is to ensure that there is no erratic behaviour or conflicts.

  • Added ability to mark episodes as watched or unwatched in the Configuration Details tab.
