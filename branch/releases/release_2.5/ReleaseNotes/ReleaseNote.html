TVSeries v2.5 Release Highlights
Skin

Blue3Wide skin refresh and Translation support

Updated Blue3Wide Skin layout, modularized the skin files so its easier to maintain and read, better visibility of the Fanart and the GUI is fully Translated. For users that dont like the new look you can find the old skin in the Zip Package.

Skin properties are now read from all TVSeries xmls files so skinners can modularize skin files.

New Skin properties, see Skin Designers Guide for more information.

Translations

Updated Translations and Publishing to skin

The following languages have been updated: Dansk, Czech, Dutch, Italian, Spanish, Greek, Russian German, French, Danish.

New Translations have been added so skinners can fully translate any existing TVSeries strings used. Translations can also be used in Skin Formatting rules. See Debug Log for a complete list of translatable strings.

Thanks to the following people whom contributed translations: 1stDead, Kucheek, HenkieFlits, RoChess, macro.tips Cybertex, Pedernal74, elapse, Undel, Rasputin, catavolt, truffe, ubela64, Steffansp.

Import / File Monitoring

Improvements to way Network Shares are handled and a new Import feature

Improved support for Network Shares when monitoring paths for changes. All mapped shares are translated to UNC paths and are now always picked by File Monitor when changes occur e.g. Delete File, Added File, Renamed File.

File Monitoring interval is now configurable and defaults to 5 mins. Setting can be changed in General Settings tab of configuration.

Added ability to update a single series/seasons/episodes from with-in the GUI. This also has the ability to delete and refresh Episode thumbnails which are sometimes out-dated or wrong. Setting for 'Update' is available from the Action menu after selecting an item.

Configuration

Speed Improvements and better Details Layout in Gridview.

Improved Speed of loading the configuration dialog, esp. for large number of episodes.

Improved Speed of Selections in Details tree, cleaned up Display Field names and removed unnecessary fields.

Fixed Configuration not displaying Episodes in details tab for large episode collections (15,000+).

Fixed Right Click select of series/seasons/episodes in Details Tree.

Delete Logic is now consistent to the behaviour in the GUI.

Logo Rules / Formatting Rules tabs are now hidden. Skinners can set import="false" in TVSeries.SkinSettings.xml to show tabs.

Subtitles

Subtitle Downloading is no longer handled internally by plugin

TVSeries now uses the SubtitleDownloader library provided by Seco, the Subtitle downloader library includes the support for many Subtitle providers/languages and can be configured in the plugin.

Installer

Installer now include some new 3rd Party Libraries

MediaInfo has been added back to package so old MediaPortal versions can easily be upgraded, current version is v0.7.27.0.

Libraries to support Subtitle downloading have been added SubtitleDownloader.dll (v1.6.0) and 7z.dll.

Database backup path has changed to the MediaPortal Database directory to make it easier to find and apply backup if needed.

General / Misc

Added new setting to Show season view when there is only one season available. Setting can be changed in General Settings of configuration.

Fixed and Re-Enabled Up/Down buttons in Parsing Expressions, much easier to add new ones and change order now.

Changed defaults for 'Prompt to Rate Episode' and 'Automatically choose Aired episode order' to enabled.

Added support 'Recently Added Episode' for new feature of the InfoService plugin (thanks edsche).

Fixed Sort Order of episodes in the 'Recently Added' view to respect Time component. Episodes now added on the same day are now sorted correctly.

Fixed a crash in Episode only views e.g. 'Recently Added' and 'Latest' when viewing an episode that did not get parsed from online.

Added new configuration setting to 'Substitute Missing Season Poster with corresponding Series Poster', default is disabled. Setting can be changed in General Settings of configuration

Episode Counts in Series/Season List will now only count episodes which have an AirDate less than or equal to Today.

TVSeries v2.3 Release Highlights
Views System

The views system has been upgraded to support several new features:

The user can now create tagged views, this allows you to add any series to a defined view e.g. Documentaries, Kids Shows, Adults Shows etc...

Tagged Views can be created when adding a series to a new view from the plugins context menu with-in MediaPortal or from the Views tab in Configuration. The Configuration program allows you to quickly multi-select a range of Series to add to a particular view using the Add/Edit view dialog.

Several new Built-in views have been added, some of which include: Actors, Adult Series, Kids Series, Continuing Series, Ended Series, Top 10 etc. These views are available for selection in the Views Configuration using the Templates dialog.

Parental Control support, you can now protect views from the configuration views tab. After you have locked down your views, enter a 4-Digit Pin in the Pin Code dialog. When a protected view is selected from the plugin in MediaPortal you will be prompted to insert your Pin Code, if the pin entered is correct you will see the contents of the protected view. If you have correctly entered a pin code and then re-enter the view you will not be prompted unless you restart/resume or manually lock views from the context menu.

Rate Series/Episodes

A few new features have been added to enhance the users experience when rating episodes or series:

Supported skins will now show a Graphical Rating dialog, this will also help you rate an episode or series by giving you a meaningful description for the score e.g. Mediocre, Good, Great. Thanks goes to the MovingPictures team for sharing the dialog.

The User can choose between a 5 Star or a 10 Star rate dialog from Configuration under General Settings.

Fixed an issue where on occasion the episode list will disappear after rating an episode.

Import Settings

The Online Data settings in configuration have been updated to provide several new features:

You can now turn off Import when starting/resuming MediaPortal, this is of particular interest to people experiencing performance issues or have slow/non-existant internet connections. To completely turn off automatic imports you also need to disable Updates.

Downloading of user ratings can be disabled, when disabled the import will be faster but your episode ratings will not be synced with the online database.

Downloading of missing artwork can be disabled, during import the plugin will check if any series or season has no artwork available. If no artwork is found then an attempt will be done to check if any is available online and download them. Import will be faster when disabling, time difference is neglible on smaller databases.

During import the plugin checks all series fanart and updates the local database with any missing or new fanarts. You can now disable this option for a faster import if you only want check for updates.

You can now check for updates for episode(s) and series from Details tab in configuration. This will only do the updates on the selected item, it will not do a full import.

Automatic download of Fanart that contain Series Names/Logos is now optional, this is turned off by default.

Installer Package

An Installer has been created to help with deployment of the plugin, it has the following features:

Support for all platforms supported by MediaPortal and is also compatible with the latest MediaPortal 1.1.0 release.

Custom directories defined in MediaPortalDirs.xml will be recognised and used for all components e.g. plugin, languages, skins etc

The installer supports upgrades for future releases.

The plugin and all of its components can be uninstalled from the Add\Remove programs in the Windows Control Panel.

During Installation the database is backed up the users documents directory: %USERPROFILE%\Documents\MP-TVSeries.

Options to remove artwork and database after an Uninstall to help keep your system clean.

Checks for multiple instances of the plugin after installation, this is to ensure that there is no erratic behaviour or conflicts.

Misc Changes and Fixes

  • Series Poster and Series Banner are now exposed to the skin as seperate properties.

  • Colours used in the List facade of skins which indicate Watched and Unavailable status have been changed to playedColor and remoteColor. The Unwatched colour uses textColor.

  • New Delete options available, user can now delete from disk, database or both. Shortcut to menu can be invoked by pressing Zero on the Remote Control. Delete menu is hidden unless enabled from with-in Configuration.

  • SMS Style Virtual Keyboard support, option can be turned on from with-in Configuration.

  • Sorting of Specials in episode view can now be disabled from with-in Configuration.

  • Hide Spoilers option has been enhanced so you can Hide Overview and/or Episode Thumbnails. When episode thumbnails are hidden the Video OSD will show the Series Poster instead. These options can be changed from with-in Configuration or the Context menu of the plugin.

  • Artwork Loading Delay can now be changed from with-in Configuration. Its recommended that defaults be left unless you know what you are doing.

  • Added ability to mark episodes as watched or unwatched in the Configuration Details tab.

  • The plugin now supports the latest MediaInfo library, this is no longer included in package. As of MediaPortal 1.0.1, MediaPortal ships mediainfo library as part of its release.

  • The Import Path section in Configuration now supports UNC path names e.g. \\server\\share

  • The Random Fanart function now cycles fanart on a configurable timer, this can be changed from the default 30secs from the Fanart Chooser window.

  • Added Import/Export of Parsing Expressions and String Replacements.

  • Added options to configure Playlists in Configuration Views tab e.g. Playlist Path. Also added new Auto-Shuffle on load property.

  • Fixed DVD order for regions using a comma as the default decimal separator, thanks freddan85

  • Fixed incorrect remository url when retrieving subtitles, thanks Gabe01

  • Fixed Now Playing OSD information not being set when playing from playlist view.

  • Fixed missing Hover image on Home Screen for Blue3 and Blue3 Wide skins.

  • Much faster episode count calculation when switching to and from 'Show all local files' in options menu.

  • Faster importing, some tasks that previously took a long time such as MediaInfo retrieval and Episode Counts are now done in the background.

Note: there have been several updates (some of which are mentioned above) to the plugin which affect the skin, as a result all skins will need to be updated. See Skin Designer's Guide for more details on what has changed. Please don't raise any issues with out dated skins.
